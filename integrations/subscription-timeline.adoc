= Keystoneサブスクリプションのタイムラインを表示する
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
NetAppコンソールのKeystoneダッシュボードには、 Keystoneサブスクリプションのタイムライン ビューが提供され、アクティベーション、変更、更新日などのイベントが表示されます。このタイムライン ビューはDigital Advisorでは使用できません。

サブスクリプションのタイムラインを表示するには、次の手順に従います。

.手順
. コンソールの左側のナビゲーション メニューから、*ストレージ > Keystone > サブスクリプション*を選択します。
. サブスクリプションのタイムラインを表示するには、「*サブスクリプション番号*」列でサブスクリプション番号を選択します。
+
*電流消費*タブにリダイレクトされます。

. *サブスクリプションタイムライン*タブを選択します。
+
image:console-subscription-timeline-graph.png["重要なイベントとともに、サブスクリプションのタイムラインを折れ線グラフ形式で表示するスクリーンショット"]

+
サブスクリプションタイムラインはデフォルトで折れ線グラフとして表示され、重要なイベントがアイコンでマークされます。異なるタイプの複数のサブスクリプション更新が同じ日に発生した場合、チャートには個別のアイコンではなく数字が表示されます。数字の上にカーソルを合わせると、その日に発生したすべての更新タイプが表示されます。また、任意のイベントアイコンの上にカーソルを合わせると、日付やコミット済み容量の変更などの詳細が表示されます。これらのイベントを表形式で表示するには、*View as*ドロップダウンから*Table*オプションを選択します。他のサブスクリプションタイムラインを表示するには、*Tracking ID*ドロップダウンから選択または検索します。

+
image:console-subscription-timeline.png["重要なイベントとともに、サブスクリプションのタイムラインを表グラフ形式で表示するスクリーンショット"]
