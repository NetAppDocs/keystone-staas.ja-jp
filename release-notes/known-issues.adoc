= 既知の問題
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
既知の問題には、Keystoneサブスクリプションサービスの効果的な利用を妨げる可能性のある問題が記載されています。

次の既知の問題がNetApp Keystone STaaSで報告されています。

[cols="3*"]
|===
| 既知の問題 | 製品説明 | 回避策 


 a| 
プライマリサブスクリプションとリンクサブスクリプションでボリュームの詳細を使用できません
 a| 
[ボリュームの概要]*タブには、プライマリサブスクリプションとリンクされたセカンダリサブスクリプションのボリュームの総数、QoS準拠ステータス、保護されているボリュームの数、消費容量の合計がゼロと表示されます。
 a| 
なし



 a| 
サブスクリプションのレポートを生成できません
 a| 
[Keystone Subscriptions]ページの*[Subscriptions]*タブで10 ~ 12個を超えるサブスクリプションを選択した場合、*[Include data for each service level]*オプションの*[Download CSV]*ボタンをクリックしてもレポートが生成されません。
 a| 
なし

|===
